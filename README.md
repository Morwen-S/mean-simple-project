# MEAN test project

[LiveDemo](https://mean-stack-exp.herokuapp.com/)

## Project setup
Server side:
```
npm install
```
Front side:
```
cd ./client && npm install
```
###MongoDB on Ubuntu
Start:
```
sudo systemctl start mongod
```
Stop: 
```
sudo service mongod stop
```
### Compiles for development
Start server: 
```
npm start
```
Start front:
```
cd ./client && ng serve
```
